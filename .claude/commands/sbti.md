# sbti

运行 SBTI 性格测试。

## 使用方式

```
/sbti [用户信息]
```

## 流程

### 第一步：题目适配

调用 `/quiz-designer SBTI --type spectrum --cases SBTI` 生成测试方案，并根据传入的**用户信息**调整：

| 用户信息 | 适配方向 |
|----------|----------|
| 职业背景 | 题目情境贴近职业场景（如程序员/设计师/销售） |
| 年龄段 | 措辞语气调整（学生/职场新人/资深人士） |
| 特定话题 | 结果页植入相关话题钩子 |
| 无传入 | 使用通用情境设计 |

### 第二步：运行测试

按设计方案运行 SBTI 状态机：

```
INIT → QUESTIONS → RESULT → DONE
```

#### 状态说明

| 状态 | 含义 | LLM 应做 |
|------|------|----------|
| `INIT` | 刚启动 | 输出开场白 + 记录当前题号 `q=1` |
| `QUESTIONS` | 答题中 | 输出当前题 + 选项，等用户回复 |
| `RESULT` | 计算结果 | 根据答案计算类型，输出完整结果页 |
| `DONE` | 完成 | 引导分享或再测 |

#### 计分规则

每维度取**多数极性**（平票时取后者，即 B 极）：

```
E > I → E，否则 → I
S > N → S，否则 → N
T > F → T，否则 → F
J > P → J，否则 → P
```

拼接即为 4 字母类型。

#### 交互识别

接受以下格式：
- 纯字母：`A` / `B` / `C` / `D`
- 带括号：`A.` / `B.` / `(C)`
- 中文：`选A` / `B吧` / `我觉得C比较对`

不区分大小写。

#### 答案记录格式

在对话中用**有序列表**记录：

```
答案记录：
1. A
2. B
3. A
...
```

### 结果页格式

```markdown
## 🎯 你的 SBTI 类型是：ENFJ

**代表角色**：《死亡诗社》基廷老师
**关键词**：领航者 / 理想主义者 / 社交蝴蝶

### 📊 维度解析

| 维度 | 你的倾向 | 说明 |
|------|----------|------|
| E/I | E（外向）| 从社交中获得能量 |
| S/N | N（直觉）| 看本质、重可能性 |
| T/F | F（情感）| 决策优先考虑人情 |
| J/P | J（判断）| 喜欢计划和控制 |

### 💬 一句话描述

> 你是一个充满感染力的理想主义者，天生擅长激励他人。
> 在团队中，你往往是那个点燃热情、凝聚共识的人。

---

✨ 测试完成！想再测一次请说"重测"
```
