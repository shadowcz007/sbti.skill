# sbti.skill

面向 **Claude Code / Claude** 的测一测内容设计技能与 **SBTI 测一测**斜杠命令。仓库不包含可独立部署的 Web/CLI 应用，而是通过 `.claude/` 下的技能与命令定义，让助手按统一规格拆解题库、计分、结果页与传播节点，并在对话中完整执行 SBTI 流程。

## 仓库结构

| 路径 | 说明 |
|------|------|
| `.claude/skills/quiz-designer/SKILL.md` | **quiz-designer**：从主题拆解可落地规格（题型谱系、维度与计分、结果页与传播） |
| `.claude/commands/sbti.md` | **sbti**：SBTI 状态机、30 题 + 特殊题、计分与人格库（25 常规 + 兜底 + 彩蛋） |

将本仓库（或上述路径）接入 Claude Code 可识别的项目配置后即可使用。

## quiz-designer

**适用**：从零设计「测一测」任一环（维度、题目、计分、结果页、传播），或输出 Markdown / JSON / 全文案。

技能将结果形式分为多类，例如：

- **人格类型型**：离散标签（如十六型）
- **光谱连续型**：连续分值或分档后再映射区间
- **人格模式匹配型**：多维分档（如每维 L/M/H）与库内标准模式比对距离（**SBTI** 见技能内案例）
- **知识答题型**：正确率 / 分数
- **数值测算型**：公式与输入变量

默认交付物：类型与维度定义、题目清单、评分说明、结果页模板、传播规划。

```text
/quiz-designer <主题> [--type ...] [--dimensions N] [--cases SBTI] [--output markdown|json|full]
```

选项与示例见 `SKILL.md`。

## sbti（SBTI 测一测）

**SBTI** 为中文互联网流传的趣味人格测试（2025 年前后自 B 站等内容侧扩散；与 MBTI 名称相近，但**不是** E/I、S/N、T/F、J/P 四轴拼字）。

**规则摘要**：

- **15 个子维度**（自我 S、情感 E、态度 A、行动驱力 Ac、社交 So，各 3 维）
- 每维 **2 题**，每题 **A/B/C = 1/2/3 分** → 每维 raw 分映射为 **L / M / H**
- 得到 **15 位**等级向量，与 **25 种常规人格**的预设模式算**总距离**，再换算 **相似度 = (1 − 总距离/30) × 100%**
- 第一名相似度 **低于 60%** → **HHHH（傻乐者）**兜底
- **特殊题**链路中选定「白酒灌保温杯当白开水喝」等选项 → **DRUNK（酒鬼）**，跳过常规匹配

```text
/sbti [用户信息]
```

状态：`INIT → QUESTIONS → RESULT → DONE`。常规题 **q1–q30**，其后为门控 **特殊题**；交互支持 `A`/`B`/`C` 及常见中文选答。

完整题库、27 种人格原文、计分细节与外链见 `.claude/commands/sbti.md`。
